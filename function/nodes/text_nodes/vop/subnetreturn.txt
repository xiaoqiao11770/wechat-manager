= Return =

"""Generates a return statement inside a method or a function defined by the parent subnet."""

== Overview ==

The Return node generates a return statement in a method or a function. This node is commonly used inside a [Method Subnet VOP|Node:vop/methodsubnet] to define the return type and provide a way to generate a statement that returns a value. The return type is determined by the type of the node wired into the return node input.
