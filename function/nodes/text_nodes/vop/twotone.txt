= Two Tone =

"""Performs simple, cartoon-like shading."""

This operator performs simple, cartoon-like shading.

If the position (`P`), normal vector (`N`) and incident vector (`I`) are not
connected, the global variables by the same names will be used instead.
If `N` is connected, the incoming vector must be normalized.
