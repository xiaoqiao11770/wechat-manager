= Negate =

"""Negates the incoming integer, float, vector or vector4 value."""

This operator negates the incoming integer, float, vector or vector4
value.

It represents a simpler version of the [Multiply Constant|Node:vop/mulconst] operator
because it does not perform a multiplication by -1.
