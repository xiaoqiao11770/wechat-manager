= Rotate by Quaternion =

"""Rotates a vector by a quaternion."""

Rotates the provided vector by the quaternion.  Since the input is treated as a vector, it is a rotation around the origin.

The quaternion should be normalized.

@related
    - [Node:vop/invert]
