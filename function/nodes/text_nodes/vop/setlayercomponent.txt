= Set Layer Component =

"""Sets a layer's components to new values."""

When any of the inputs are connected, the incoming layer's component of the same name is set to the input's value.

@inputs

layer:
    The layer to be modified.

F:
    #contentfrom: layerpack#F

Of:
    #contentfrom: layerpack#Of

Ce:
    #contentfrom: layerpack#Ce

P:
    #contentfrom: layerpack#P

N:
    #contentfrom: layerpack#N

@outputs

layer_out:
    The modified layer.

- [Node:vop/layermix]
- [Node:vop/layerpack]
