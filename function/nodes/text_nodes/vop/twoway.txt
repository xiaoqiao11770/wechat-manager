= Two Way Switch =

"""Takes an integer input."""

This operator takes an integer input. If this input value satisfies the
__Condition__ parameter, the output is set equal to the first input value.
If the integer input does not satisfy the __Condition__ parameter, the
output is set equal to the second input value. This provides a simple
way to switch between two values and simulate an if-then-else construct.
