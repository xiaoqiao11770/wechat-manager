= Arctangent =

#type: node
#context: vop
#internal: atan
#icon: VOP/atan

"""Performs the atan2() function"""

This operator takes an x and y coordinate and computes the arctangent of the angle to the (x, y) coordinate.

The resulting value is in the range -PI to PI.
