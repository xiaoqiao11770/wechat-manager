= Volume VOP Output Variables =

""" Simple output variable for Volume VOP Networks."""

This is a simple place holder to write out the density volume.  If you
dive inside you will see a [Parameter VOP|Node:vop/parameter] set to write out the density
attribute.  This can be used as your prototype for how to write out
other volumes.
