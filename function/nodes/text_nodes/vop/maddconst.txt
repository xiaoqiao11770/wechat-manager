= Multiply Add Constant =

"""Will take the input value, add the pre-add amount, multiply by the
constant multiplier, then add the post-add amount."""

This operator will take the input value, add the pre-add amount,
multiply by the constant multiplier, then add the post-add amount.

The incoming value may be an integer, float, vector or vector4 value.
The output type matches the input type.
