= RSL Material =

"""Implements an RSL material."""

This node implements an RSL material for RIS mode. While RIS mode expects BxDFs
as shader assignment, it can also accept traditional RSL shaders too. This
shader node provides a way to define shaders for RIS using the RSL approach.
