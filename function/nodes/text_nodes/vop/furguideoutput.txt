= Fur Guide Output Variables and Parameters =

#type: node
#context: vop
#internal: furguideoutput
#icon: SOP/fur

"""Provides inputs representing the output variables of a fur guide shader
network."""

This operator provides inputs that represent all the output variables
for fur guide shader network. There are no outputs to this operator.

@parameters

== Inputs ==

    Point Position:
	Position of each vertex in the generated hairs.

    Width:
	Width of the rendered curves.

== Outputs ==

    None.

@related

- [Node:vop/furskinglobal]
- [Node:vop/furskinoutput]
- [Node:vop/furguideglobal]
