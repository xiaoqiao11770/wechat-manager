= CHOP Input =

"""Returns a sample value in one of the 4 input CHOPs connected to the VEX CHOP."""

This operator returns a sample value in one of the 4 input CHOPs connected to the VEX CHOP. The sample may be in any channel, at any time.

If the queried sample is floating point, it is linearly interpolated.

@related
    - [Node:vop/isconnected]
