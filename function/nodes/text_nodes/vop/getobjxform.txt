= Get Object Transform =

"""Gets the transform matrix of a named object in camera (current) space."""

TIP:
    The [Transform VOP|Node:vop/transform] is more flexible and useful in many circumstances.

@inputs

Object name:
    The path of an object-level node.

@outputs

Object transform:
    The transformation matrix of the object in camera space.

@related

- [Node:vop/transform]
