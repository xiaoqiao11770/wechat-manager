= Vector4 to Vector =

"""Converts a vector4 to a vector and also returns the fourth component
of the vector4."""

This operator converts a vector4 to a vector and also returns the fourth
component of the vector4. The three components of the resulting vector
can be used to represent positions, directions, normals or colors (RGB
or HSV).
