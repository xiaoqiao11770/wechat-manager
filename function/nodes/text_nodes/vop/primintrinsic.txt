= Primitive Intrinsic =

"""Evaluates an intrinsic on a given primitive."""

This operator evaluates an intrinsic for a given primitive.

If the operation fails, the result is 0.

It will fail if the primitive doesn't have an intrinsic of that name, or the intrinsic is of the wrong type.
