= View =

"""Tumble, track, rotate, and zoom the view."""

This tool lets you tumble, track, rotate, and zoom the view without
holding the Space bar.

Mouse Controls:

table>>
    tr>>
        th>> Mouse
        th>> No modifier
        th>>
            ((Alt))
        th>>
            ((Ctrl))
    tr>>
        td>>
            ((LMB))
        td>> Tumble
        td>>
        td>> Rotate
    tr>>
        td>>
            ((MMB))
        td>> Dolly
        td>> Zoom
        td>> Box zoom
    tr>>
        td>>
            ((RMB))
        td>> Pan
        td>>
        td>> Menu
Use ((RMB)) to show a list of view-related commands available in
this tool.
