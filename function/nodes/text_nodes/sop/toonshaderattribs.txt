= Toon Shader Attributes =

"""Sets attributes used by the [Toon Color Shader|Node:shop/tooncolorshader] and [Toon Outline Shader|Node:shop/toonoutlineshader]."""

This node let's you set toon shader parameters for a group of primitives. Use this to color cartoon objects without using textures.

@parameters

Group:
    #id: group
    The group to modify.

Highlight Color:
    #id: colorhigh
    Sets the color of [Toon Color Shader|Node:shop/tooncolorshader] highlights.

Mid Color:
    #id: colormid
    Sets the color of [Toon Color Shader|Node:shop/tooncolorshader] lit areas.

Shaded Color:
    #id: colorlow
    Sets the color of [Toon Color Shader|Node:shop/tooncolorshader] shaded areas.

Outline Scale:
    #id: outlinescale
    Scaled the thickness set by the [Toon Outline Shader|Node:shop/toonoutlineshader]'s thickness parameters.
