= Empty Data =

""" Creates an Empty Data for holding custom information. """

The Empty Data DOP creates an empty piece of data on an object. The
purpose of this empty data is to contain custom information about an
object. This custom information will most often be created and
controlled by the [Modify Data DOP|Node:dop/modifydata] or the [Script
Solver DOP|Node:dop/scriptsolver].
