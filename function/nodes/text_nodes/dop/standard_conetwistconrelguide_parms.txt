#type: include
Show Softness Threshold:
    Show where the constraint begins to take effect. This is only used if __Softness__ is greater than 0 and less than 1.
Color:
    The color of the primary guide geometry.
Secondary Color:
    The color of the secondary guide geometry. This includes the __Softness Threshold__ and the indicator of the current twist.
Guide Size:
    Scales the guide geometry.
Show Object Link:
    This parameter controls the display of guide geometry
    connecting the constraint to the constrained object.
