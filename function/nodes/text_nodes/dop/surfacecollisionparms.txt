= Surface Collision Parameters =

"""Control the thickness of the object that collides with cloth."""

These parameters live inside the [Static Object|Node:dop/staticobject]. They are used to control the thickness of the object that collides with cloth. This is very useful if you want to have some distance between your static object and your cloth.

@parameters

Thickness:
    Control the thickness of the object that collides with cloth.
