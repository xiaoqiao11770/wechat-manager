= Soft Body Collision Properties =

"""Defines how a soft body object responds to collisions."""

The Soft Body Collision Properties DOP defines how a soft body object, such as a cloth object or wire object, responds to collisions and contacts. It is applied automatically as subdata to
a [Cloth Object|Node:dop/clothobject] and [Wire Object|Node:dop/wireobject].
