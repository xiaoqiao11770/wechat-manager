= VEX Shadow Shader =

"""This network defines a SHOP operator type."""

This network defines a SHOP operator type. This SHOP operator can be
used as a shadow shader when rendering to Mantra.

@parameters
    SHOP Type Name:
        The name of the new operator type defined by this network.
    Show Operator in Tab Menu:
        If this option is turned on you will be able to access your shader from the [Tab Menu|/basics/tabmenu] in the SHOP network.

@related
    - [Node:vex/surface]
    - [Node:vex/light]
