= VOP VEX Photon =

"""Container for a VEX photon shader implementation."""

This SHOP provides an container for a VOP implementation of a VEX photon
shader.

@parameters

Compiler:
    Specifies the command line used to compile the VEX code.
Force Compile:
    Compiles the VEX code.
