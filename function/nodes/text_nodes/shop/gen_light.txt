= Generic Light =

"""Implements a generic light shader."""

This SHOP implements a generic light shader. It provides input fields
for every renderer supporting light shading.

To see what light shaders are available for each renderer, click the "+"
button beside the input field corresponding to each renderer, then hit
"Accept" in the Dialog Script to dump the shader string into the
matching input field.
