= Generic Displacement =

"""Implements a generic displacement shader."""

This SHOP implements a generic displacement shader. It provides input
fields for every renderer supporting displacement shading.

To see what displacement shaders are available for each renderer, click
the "+" button beside the input field corresponding to each renderer,
then hit "Accept" in the Dialog Script to dump the shader string into
the matching input field.
