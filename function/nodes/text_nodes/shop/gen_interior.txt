= Generic Interior =

"""Implements a generic interior fog shader."""

This SHOP implements a generic interior fog shader. It provides input
fields for every renderer supporting interior fog shading.

To see what interior fog shaders are available for each renderer, click
the "+" button beside the input field corresponding to each renderer,
then hit "Accept" in the Dialog Script window to dump the shader string
into the matching input field.
