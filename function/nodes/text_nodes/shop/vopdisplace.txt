= VOP VEX Displacement =

"""Container for a VEX displacement shader implementation."""

This SHOP provides an container for a VOP implementation of a VEX
displacement shader.

@parameters

Compiler:
    Specifies the command line used to compile the VEX code.
Force Compile:
    Compiles the VEX code.
