= VOP VEX Image3D =

"""Container for a VEX image3d shader implementation."""

This SHOP provides an container for a VOP implementation of a VEX image3d
shader.

@parameters

Compiler:
    Specifies the command line used to compile the VEX code.
Force Compile:
    Compiles the VEX code.
