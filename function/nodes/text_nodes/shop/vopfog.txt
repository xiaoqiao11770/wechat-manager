= VOP VEX Fog =

"""Container for a VEX fog shader implementation."""

This SHOP provides an container for a VOP implementation of a VEX fog shader.

@parameters

Compiler:
    Specifies the command line used to compile the VEX code.
Force Compile:
    Compiles the VEX code.
