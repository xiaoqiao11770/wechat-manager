= Generic Surface =

"""Implements a generic surface shader."""

This SHOP implements a generic surface shader. It provides input fields
for every renderer supporting surface shading.

To see what surface shaders are available for each renderer, click the
"+" button beside the input field corresponding to each renderer, then
hit "Accept" in the Dialog Script to dump the shader string into the
matching input field.
