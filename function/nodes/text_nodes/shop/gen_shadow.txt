= Generic Shadow =

"""Implements a generic shadow shader."""

This SHOP implements a generic shadow shader. It provides input fields
for every renderer supporting shadow shading.

To see what shadow shaders are available for each renderer, click the
"+" button beside the input field corresponding to each renderer, then
hit "Accept" in the Dialog Script to dump the shader string into the
matching input field.
