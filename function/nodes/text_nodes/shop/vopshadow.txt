= VOP VEX Shadow =

"""Container for a VEX shadow shader implementation."""

This SHOP provides an container for a VOP implementation of a VEX shadow
shader.

@parameters

Compiler:
    Specifies the command line used to compile the VEX code.
Force Compile:
    Compiles the VEX code.
