= VOP VEX Light =

"""Container for a VEX light shader implementation."""

This SHOP provides an container for a VOP implementation of a VEX light
shader.

@parameters

Compiler:
    Specifies the command line used to compile the VEX code.
Force Compile:
    Compiles the VEX code.
