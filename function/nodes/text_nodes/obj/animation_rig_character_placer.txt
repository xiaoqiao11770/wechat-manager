= Animation Rig Character Placer =

"""Provides parameters to animate the character placer."""

@parameters

Display Bodypart:
     Toggles the display of character placer.
    
== Animation ==

Control LOD:
    Changes the level of display for the rig.

Set Animation Defaults:
    Sets the animation parameter values to their default values.
    
Hook Object:
    The parent object for the rig.

Translate/Rotate/Scale:
    Translates, rotates, and scales the rig.
    
== Preferences ==

Display Animation Controls:
    Toggles the display of animation controls on the rig.

Control Scale:
    Modify the per-part control scale.

Control Color:
    Modify the per-part control color.

=== Control Level of Detail ===

Character Placer:
    Modify the display of the character placer.
