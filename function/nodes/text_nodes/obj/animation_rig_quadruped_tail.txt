= Animation Rig Quadruped Tail =

"""Provides parameters to animate the tail for a quadruped character."""

@parameters

Display Bodypart:
     Toggles the display of proxy geometry for the tail.
    
== Animation ==

Control LOD:
    Changes the level of display for the tail controls.

Set Animation Defaults:
    Sets the animation parameter values to their default values.
    
The rest of these parameters provide controls to translate and rotate the parts of the tail.

== Rigging ==

=== Hooks ===

Hook Object:
    The path of the hook object.
    
    NOTE:
        The hook object provides parent transforms to the body part. 
    
=== Pretransforms ===

Provides controls to change the pretransform positions of the parts of the tail.

=== Hook Out Rotations ===

Provides controls to set the hook out rotations for the parts of the tail.

== Preferences ==

Display Proxy Geometry:
    Toggles the display of the proxy geometry for the body part.

Display Animation Controls:
    Toggles the display of animation controls on the rig.

Control Scale:
    Modify the per-part control scale.

Control Color:
    Modify the per-part control color.

=== Control Level of Detail ===

These parameters allow you to modify the display of the parts of the tail.
