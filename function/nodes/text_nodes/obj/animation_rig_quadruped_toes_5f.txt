= Animation Rig Quadruped Toes =

"""Provides parameters to animate the toes for a quadruped character."""

@parameters

Display Bodypart:
     Toggles the display of proxy geometry for the toes.
    
== Animation ==

Control LOD:
    Changes the level of display for the toes controls.

Set Animation Defaults:
    Sets the animation parameter values to their default values.
    
=== Thumb ===

Provides controls to translate and rotate the three sections of the thumb.

=== Index ===

Provides controls to translate and rotate the three sections of the index toe.

=== Middle ===

Provides controls to translate and rotate the three sections of the middle toe.

=== Ring ===

Provides controls to translate and rotate the three sections of the ring toe.

=== Pinky ===

Provides controls to translate and rotate the three sections of the pinky toe.

== Rigging ==

Thumb/Toe Hook Object:
    The path of the hook object for the thumb/toe.
    
    NOTE:
        The hook object provides parent transforms to the body part. 

Side:
    Determines if the hand is the left or right hand of the character.
    
=== Pretransforms ===

Provides controls to change the pretransform positions for the parts of the toes.

=== Deform Rotations ===

Provides controls to change the deform rotations of the toes.

=== Hook Out Rotations ===

Provides controls to set the hook out rotations for the parts of the toes.

== Preferences ==

Display Animation Controls:
    Toggles the display of animation controls on the rig.

Display Proxy Geometry:
    Toggles the display of the proxy geometry for the body part.

Control Scale:
    Modify the per-part control scale.

Control Color:
    Modify the per-part control color.
    
=== Control Level of Detail ===

These parameters allow you to modify the display of the thumb, indexes, middles, rings, and pinkies.
