= Color Wheel =

"""Generates a simple HSV color wheel."""

This COP generates a simple HSV color wheel. It is useful in visualizing
various color correction operations.

[Include:is_a_generator]

@parameters

== Color Wheel ==

Scale:
    Scale the color wheel.
Saturation at Center:
    Color saturation in the middle of the wheel.
Saturation at Edge:
    Color saturation at the edge of the wheel.
Value at Center:
    Color value in the middle of the wheel.
Value at Edge:
    Color value at the edge of the wheel.
